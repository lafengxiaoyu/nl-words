# 🇳🇱 荷兰语单词学习应用

一个现代化的荷兰语单词学习应用，使用 React + TypeScript + Vite 构建。

## 功能特性

- ✨ **单词卡片翻转** - 点击卡片查看中文翻译
- 📊 **学习进度追踪** - 实时显示学习进度和统计
- ✅ **掌握标记** - 标记已掌握的单词
- 🔀 **随机排序** - 打乱单词顺序，增加学习挑战
- 💾 **数据持久化** - 自动保存学习进度到本地存储
- 📱 **响应式设计** - 完美适配手机和电脑

## 快速开始

### 1. 安装依赖

```bash
cd web
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

启动后，浏览器会自动打开，或者手动访问显示的地址（通常是 `http://localhost:5173`）

### 3. 构建生产版本

```bash
npm run build
```

构建完成后，文件会在 `dist` 目录中。

### 4. 预览生产版本

```bash
npm run preview
```

### 5. 代码检查

```bash
npm run lint
```

## 使用说明

### 基本操作

1. **查看单词**：页面中央显示荷兰语单词卡片
2. **翻转卡片**：点击卡片查看中文翻译
3. **切换单词**：使用"上一个"和"下一个"按钮浏览单词
4. **标记掌握**：点击"标记掌握"按钮标记已学会的单词
5. **查看进度**：顶部进度条显示学习进度

### 高级功能

- **随机排序**：点击"🔀 随机排序"打乱单词顺序
- **查看统计**：点击"📊 显示统计"查看详细学习数据
- **重置进度**：点击"🔄 重置进度"清除所有学习记录

## 测试功能

### 功能测试清单

1. ✅ **卡片翻转**
   - 点击卡片，应该看到翻转动画
   - 正面显示荷兰语，背面显示中文

2. ✅ **单词导航**
   - 点击"下一个"按钮，应该切换到下一个单词
   - 点击"上一个"按钮，应该切换到上一个单词
   - 到达最后一个单词后，点击"下一个"应该回到第一个

3. ✅ **掌握标记**
   - 点击"标记掌握"按钮，按钮文字变为"✓ 已掌握"
   - 进度条应该更新
   - 单词信息处显示"已掌握"徽章

4. ✅ **进度追踪**
   - 标记单词后，顶部进度条应该更新
   - 进度百分比应该正确计算

5. ✅ **数据持久化**
   - 标记几个单词为已掌握
   - 刷新页面，进度应该保持不变

6. ✅ **随机排序**
   - 点击"随机排序"，单词顺序应该改变
   - 当前索引应该重置为 0

7. ✅ **统计功能**
   - 点击"显示统计"，应该显示统计面板
   - 统计数据应该正确显示总单词数、已掌握数等

8. ✅ **重置功能**
   - 点击"重置进度"，应该弹出确认对话框
   - 确认后，所有单词的掌握状态应该被清除

### 浏览器测试

建议在以下浏览器中测试：

- ✅ Chrome/Edge (推荐)
- ✅ Firefox
- ✅ Safari
- ✅ 移动端浏览器

### 响应式测试

- 调整浏览器窗口大小，检查布局是否正常
- 在手机浏览器中打开，测试移动端体验

## 技术栈

- **React 19** - UI 框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **CSS3** - 样式和动画

## 项目结构

```
web/
├── src/
│   ├── App.tsx          # 主应用组件
│   ├── App.css          # 应用样式
│   ├── main.tsx         # 入口文件
│   ├── index.css        # 全局样式
│   └── data/
│       └── words.ts     # 单词数据
├── public/              # 静态资源
├── package.json         # 项目配置
└── vite.config.ts       # Vite 配置
```

## 自定义单词库

编辑 `src/data/words.ts` 文件可以添加或修改单词：

```typescript
export const words: Word[] = [
  { id: 1, dutch: "hallo", chinese: "你好", mastered: false },
  // 添加更多单词...
];
```
